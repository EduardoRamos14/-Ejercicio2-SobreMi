Lista Descriptiva

Función:

En programación, una función es un bloque de código que realiza una tarea específica cuando es llamado o invocado. Las funciones ayudan a organizar el código en fragmentos reutilizables y modulares. Pueden aceptar parámetros como entrada y devolver un valor como salida.

let:

let es una palabra clave utilizada en JavaScript para declarar una variable. Las variables declaradas con let están limitadas al bloque, declaración o expresión donde se definen. Esto significa que la variable solo es accesible dentro de su ámbito de declaración. Además, las variables declaradas con let pueden ser actualizadas, pero no redeclaradas en el mismo ámbito.

const:

const es otra palabra clave utilizada en JavaScript para declarar una variable. Las variables declaradas con const también están limitadas al bloque, declaración o expresión donde se definen. La diferencia principal entre let y const es que las variables declaradas con const no pueden ser reasignadas después de su inicialización. Es decir, una vez que se asigna un valor a una variable const, no se puede cambiar posteriormente.

<code>:

La etiqueta <code> se utiliza para mostrar fragmentos de código en un documento HTML. El texto dentro de esta etiqueta se muestra en una fuente de ancho fijo (generalmente tipo de letra monoespaciado), lo que significa que mantiene el formato y los espacios en blanco tal como están escritos en el código fuente original.

<div>:

La etiqueta <div> es un contenedor genérico en HTML que se utiliza para agrupar otros elementos y aplicar estilos mediante CSS. Se utiliza para dividir el contenido de la página en secciones o grupos lógicos. Los elementos dentro de un <div> pueden incluir texto, imágenes, formularios, y otros elementos HTML.

<button>:

La etiqueta <button> se utiliza para crear un botón en un formulario HTML. Los botones son interactivos y permiten a los usuarios realizar acciones, como enviar un formulario, realizar una operación en JavaScript, o navegar a otra página web. Los botones pueden contener texto, imágenes u otros elementos HTML y pueden asociarse con eventos para realizar acciones específicas cuando se hace clic en ellos.

Lista Anidada

  • Mamíferos:
    • Los mamíferos son animales vertebrados.
    • Tienen glándulas mamarias que les permiten alimentar a sus crías con leche.
    • Suelen tener pelo o pelo en alguna etapa de su vida.
    • Dan a luz a crías vivas en lugar de poner huevos.
    • Ejemplos:
      • Perros
      • Gatos
      • Ballenas
      • Humanos
  • Aves:
    • Las aves son animales vertebrados con plumas.
    • Tienen un pico y patas con garras.
    • Ponen huevos y incuban sus crías hasta que pueden valerse por sí mismas.
    • Tienen sistema respiratorio altamente eficiente, incluyendo sacos aéreos.
    • Ejemplos:
      • Águilas
      • Pingüinos
      • Canarios
  • Reptiles:
    • Los reptiles son animales vertebrados de sangre fría.
    • Tienen escamas en su cuerpo.
    • Ponen huevos con cáscara dura en lugar de dar a luz a crías vivas.
    • Tienen un sistema respiratorio pulmonar.
    • Ejemplos:
      • Serpientes
      • Cocodrilos
      • Tortugas
  • Peces:
    • Los peces son animales acuáticos vertebrados.
    • Tienen branquias que les permiten respirar oxígeno del agua.
    • La mayoría de los peces ponen huevos.
    • Tienen aletas que les ayudan a nadar.
    • Ejemplos:
      • Truchas
      • Tiburones
      • Peces payaso
